# Opioid
The OP: Live After Dark Bot

We were told to make our own server with our own mod and our own rules. 

So we did. 


# Development
The following variables need to be set in a .env 

```
DISCORD_TOKEN=##########
CLIENT_ID=##############
GUILD_ID=###############
``` 

To run in dev mode, use `npm run dev` after using `npm install` 

Create a dev bot on the Discord Developer Portal, feel free to call it whatever you want, it shouldn't affect anything, but I use `opioid-dev`.
